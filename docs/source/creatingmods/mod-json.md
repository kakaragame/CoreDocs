# Creating and understanding the mod.json


|Key| Description  |
|--|--|
| main-class  | The main class to the mod |
| name | the name of your mod |
| version | the version of your mod |


```json
{
  "main-class": "org.kakara.kvanilla.KVanillaMain",
  "name": "KVanilla",
  "version": "1.0-SNAPSHOT"
}
```
